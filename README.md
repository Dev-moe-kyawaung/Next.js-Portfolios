# Moe-Kyaw-Aung-Portfolio V02

A production-ready Next.js 15 portfolio website for **Moe Kyaw Aung**, an Android Senior Developer.

## Features
- Next.js App Router
- SEO metadata with `generateMetadata`
- Dynamic `robots.txt` and `sitemap.xml`
- Responsive Bootstrap-based glasspunk design
- Dark/light mode toggle
- Animated hero and reveal effects
- Contact form and FAQ
- Burmese language support
- Vercel deployment-ready

## Tech Stack
- Next.js 15
- React 19
- Bootstrap 5
- Framer Motion
- Google Fonts
- Next.js Metadata API

## Folder Structure
- `app/` — routes, layouts, metadata files
- `components/` — reusable UI sections
- `public/` — images, favicon, OG image, resume
- `next.config.js` — image configuration
